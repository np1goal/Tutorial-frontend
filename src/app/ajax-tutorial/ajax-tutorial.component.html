Introduction

AJAX is not a programming language. It is a technique for accessing web servers from a web page. AJAX stands for Asynchronous Javascript XML.

AJAX used combination of:

  • A browser built-in XMLHttpRequest object - to request data from web server
  • Javascript and HTML SOM - to display or use data

NOTE: AJAX applicationsmight use XML to transport data, but it is equally common to transport JSON data.

AJAX allows web pagesto be updated asynchronously by exchanging data with a web server behind the scenes, meaning, it is possible to update parts of a web page, wuthout reloading entire page.

XMLHttp

All Modern web browsers support XMLHttpRequest.

Create XMLHttpRequest Object

var xhttp = new XMLHttpRequest();

Access Across Domains

  • For security reasons, modern web browsers do not support access across domains.
  • So, both web page and XML file must be on the same server.

Old IE Browsers

var iexhttp = new ActiveXObject("Microsoft.XMLHTTP");

XMLHttpRequest Object Methods

Methods Description
new XMLHttpRequest() Creates new XMLHttpRequest object
abort() Cancels the current request
getAllResponseHeaders() Returns Header information
getResponseHeader() Returns specific Header information
open(method, url, async, user, psw) Specifies the request
  • method: request type - GET or POST
  • url: the file location
  • async: true(asynchronous) or false(synchronous)
  • user: optional username
  • psw: optional password
send() Sends request to server (used for GET requests)
send(string) Sends request to server (used for POST requests)
setRequestHeader(header, value) Adds a label/value pair to the header to be sent (header name, header value)

XMLHttpRequest Object Properties

Property Description
onreadystatechange Defines a fucntion to be called when the readyState property changes.
readyState Holds state of XMLHttpRequest
  • 0: request not initialized
  • 1: server connection established
  • 2: request received
  • 3: processing request
  • 4: request finished and response received
responseText Returns response data as string.
responseXML Returns response data as XML.
status Returns status-number of the request
  • 200: "OK"
  • 403: "Forbidden"
  • 404: "Not Found"
statusText Returns status text like "OK"

Request

To send a request to a server, open() and send() methods are used.

xhttp.open("GET", "ajax_info.text", true);
xhttp.send();

GET Requests

xhttp.open("GET", "demo_get.asp", true);
xhttp.send();

Exception: The above example, may give a cached data.

A unique id can be added to the URL to avoid cached data

xhttp.open("GET", "demo_get.asp?t=" + Math.random(), true);
xhttp.send();

To send information with GET method, add information to URL.

xhttp.open("GET", "demo_get2.asp?fname=Henry&lname=Ford", true);
xhttp.send();

POST Requests

xhttp.open("POST", "demo_post.asp", true);
xhttp.send();

To POST data like an HTML form, add an HTTP header with setRequestHeader(). Specify the data you want to send in the send() method.

xhttp.open("POST", "demo_get.asp", true);
xhttp.setRequestHeader("Content-type", "application/x-www-form-urlencoded");
xhttp.send();

To send information with GET method, add information to URL.

xhttp.open("GET", "demo_get2.asp?fname=Henry&lname=Ford", true);
xhttp.send();

The url

The url parameter of open() method, is an address to a file on a server. The file can be of any kind like .txt or .xml or a scripting file like .asp and .php

The Asynchronous / Synchronous

Server requests should be sent asynchronously. Sp, the Javascript does not wait or server response and perform other scripts while waiting for response and deal with response when ready.

Synchronous request is used for testing. It does not require onreadystatechange property.

Response

readyState property holds the status of the XMLHttpRequest. onreadystatechanged property defines a function to be executed when the readyState changes. onreadystatechanged property is called everytime the readystate changes.

when readystate is 4 and status is 200, the response is ready.

xhttp.onreadystatechange = function() {
  if (this.readyState == 4 && this.status == 200) {
    document.getElementById("demo").innerHTML = this.responseText;
  }
};
