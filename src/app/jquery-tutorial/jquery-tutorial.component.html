jQuery

Introduction

jQuery is a Javascript library. jQuery also simplifies a lot of complicated things from Javascript, like AJAX calls and DOM manipulation.

To use jQuery:

  • Download jQuery library from jQuery.com OR
  • Include jQuery from CDN, like Google or Microsoft

Use <script> tag to include cdn link or file path to src attribute inside <head> tag.

For Google use <script> src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"<script>

For Microsoft use <script> src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"<script>

When downloading jQuery library, there are two versions:

  • Production version - minified and compressed for live website
  • Deveopment version - for testing and development purpose (readable code)

jQuery fucntions can be written in another file with .js extension and instialized in src attribute of script tag.

Syntax

To apply jQuery, select html elements and perform action on them

$(selector).action()

Selectors

All CSS selectors can be used to select an element.

Document ready

The $document.ready( function() { // jQuery methods }); allows us to execute a fucntion when the document is fully loaded. This is to prevent any jQuery code from running before the body of the document is loaded.

on() Method

The on() method attached one or more event handlers for the selected events.

$("p").on({
    mouseenter: function() {
      $(this).css("background-color", "lightgray");
    },
    mouseleave: function() {
      $(this).css("background-color", "lightblue");
    },
    click: function(){
      $(this).css("background-color", "yellow");
    }
  });
