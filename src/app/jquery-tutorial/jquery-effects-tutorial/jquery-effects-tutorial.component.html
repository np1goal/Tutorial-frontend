jQuery

Hide/Show

hide() and show() methods

$(selector).hide(speed, callback); $(selector).show(speed, callback);

The optional speed specifies time in milliseconds or keywords like "slow" or "fast". The optional callback function is to be executed after the hide() and show() methods.

toogle() method

$(selector).toggle(speed, callback);

toggle() method jumps between hide and show method.

Fade

fadeIn() method

fadeIn() method is used to fade in a hidden element.

$(selector).fadeIn(speed, callback);

The optional speed specifies time in milliseconds or keywords like "slow" or "fast". The optional callback function is to be executed after the fadeIn() method.

fadeOut() method

fadeOut() method is used to fade in a hidden element.

$(selector).fadeOut(speed, callback);

The optional speed specifies time in milliseconds or keywords like "slow" or "fast". The optional callback function is to be executed after the fadeOut() method.

fadeToogle() method

$(selector).fadeToggle(speed, callback);

fadeToggle() method jumps between fadeIn() and fadeOut() method.

fadeTo() method

$(selector).fadeTo(speed,opacity,callback);

The jQuery fadeTo() method allows fading to a given opacity (value between 0 and 1). The optional speed specifies time in milliseconds or keywords like "slow" or "fast". The optional callback function is to be executed after the fadeTo() method.

Slide

slideDown() method

slideDown() method is used to fade in a hidden element.

$(selector).sideDown(speed, callback);

The optional speed specifies time in milliseconds or keywords like "slow" or "fast". The optional callback function is to be executed after the slideDown() method.

slideUp() method

slideUp() method is used to fade in a hidden element.

$(selector).slipdeUp(speed, callback);

slideToggle() method

$(selector).slideToggle(speed, callback);

slideToggle() method jumps between slideDown() and slideUp() method.

Animate

stop()

The jQuery stop() method is used to stop an animation or effect before it is finished. The stop() method works for all jQuery effect functions, including sliding, fading and custom animations.

$(selector).stop(stopAll,goToEnd);

The optional stopAll parameter specifies whether also the animation queue should be cleared or not. Default is false, which means that only the active animation will be stopped, allowing any queued animations to be performed afterwards. The optional goToEnd parameter specifies whether or not to complete the current animation immediately. Default is false.

Callback

A callback function is executed after the current effect is 100% finished.

$(selector).hide(speed,callback);
