Advanced Selectors

Selector Example Description
.class1.class2 .name1.name2 Selects all elements with both name1 and name2 within its class attributes.
.class1 .class2 .name1 .name2 Selects all elements with name2 that is descendant of name1.
element.class p.intro Selects all p elements with class="intro"
element1, element2 div, p Selects all div and p elements
element1 element2 div p Selects all p elements in all div elements.
element1 > element2 div > p Selects all p elements where parent is div element.
element1 + element2 div + p Selects all p elements that are placed immediately after div elements.
element1 ~ element2 ul ~ p Selects all ul elements that are preceded by p elements.

Attribute Selectors

Selector Example Description
[attribute] [target] Selects all elements with target attribute
[attribute=value] [target=_blank] Selects all elements with target=_blank
[attribute~=value] [title~=flower] Selects all elements with a title attribute containing the word "flower"
[attribute|=value] [lang|=en] Selects all elements with a lang attribute value starting with "en"
[attribute^=value] a[href^="https"] Selects every a element whose href attribute value starts with "href"
[attribute$=value] a[href$=".pdf"] Selects every a element whose href attribute value ends with ".pdf"
[attribute*=value] a[href*="w3schools"] Selects every a element whose href attribute value contains a substring of "w3schools"

Margin and Padding

Margin

  • margin: top right bottom left
  • Margin can be set to auto for divs to make is horizontally center
  • Top and bottom margins of two elements placed on top of the other, sometimes collapse into a single margin which is equal to the largest of two margins.

Padding

  • padding: top right bottom left OR
  • padding: top right bottom OR
  • padding: top-bottom right-left
NOTE: If div has width = 200px; margin = 50px; then the actual width of the div would be 200 + 50 + 50 = 300px

Flex box vs Grid

Flex box Layout is in 1-dimension - it works with row or column.

Grid Layout is in both 1-dimension and 2-dimensions - it works with both row and column.

CSS Grids

Display

display: grid; OR display: inline-grid;

Gaps

grid-column-gap: column - Gaps between columns

grid-row-gap: (row) - Gaps between rows

grid-gap: (row) (column) - Gaps between rows and columns

Lines

Templates to create rows and columns + Grid items

The lines between columns are called column lines and between rows are called row lines.

grid-column-start: 1; grid-column-end:3 - Add an item at column 1 and end at column 3

grid-column: 1/ 3 - Add an item at column 1 and end at column 3

grid-column: 2/ span 3 - Add an item at column 2 and span 3 columns (end at 4th column)

grid-row-start: 1; grid-row-end:3 - Add an item at row 1 and end at row 3

grid-row: 1/ 3 - Add an item at row 1 and end at row 3

grid-row: 2/ span 3 - Add an item at row 2 and span 3 rows (end at 4th row)

grid area: grid-row-start / grid-column-start / grid-row-end / grid-column-end

Grid containers

grid-template-rows: Create number of rows with either size or auto

grid-template-columns: Create number of columns with either size or auto

justify-content property is used to align the whole grid inside the container Horizontally.

jusity-content: space-evenly; - equal amount of space between and around columns

jusity-content: space-around; - equal amount of space around columns

jusity-content: space-between; - equal amount of space between columns

jusity-content: center; - in the middle of the container

jusity-content: start; - in the start of container

jusity-content: end; - in the end of container

align-content property is used to align the whole grid inside the container Vertically.

align-content: space-evenly; - equal amount of space between and around rows

align-content: space-around; - equal amount of space around rows

align-content: space-between; - equal amount of space between rows

align-content: center; - in the middle of the container

align-content: start; - in the top of container

align-content: end; - in the bottom of container
