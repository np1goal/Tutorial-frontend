 CSS Animations

Transforms

Transform is used to move, rotate, scale and skew elements.

translate(), translateX(), translateY() moves the element in x and y direction. +ve is forward, -ve is backward.

scale(), scaleX(), scaleY() changes width and height of the element in x and y direction. >1 is increase, <1 is decrease.

rotate() rotates the element. +deg is forward rotate, -deg is backward rotate.

skew(), skewX(), skewY() slants the element with respect to width and height direction.

matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY()) combines all the 2D transform methods into one.

NOTE: More than one transform functions can be used at a time.

Transitions

Transition allows to change property values smoothly in a duration time.

transition: width 1s changes the property width in 1sec

NOTE: More than one property can be added.

transition-timing-function property specifies the speed curve of transition effect

  • linear same speed from start to end
  • ease slow start, then fast and end slowly
  • ease-in slow start
  • ease-out slow end
  • ease-in-out slow start, slow end

Keyframes

Key frames decides the change in time frame

@keyframes drive {
  from { transform: translateX(0px); }
  to { transform: translateX(200px); }
}
.element {
   animation: drive 2s;
}

Element drives 200px from 0px.

Instead of from and to percentages can also be used like 0%, 25%, 75% and 100%

Fill mode

animation-fill-mode: property that specifies a style of an element when the animation is not playing.

  • forwards: element will retain the value of last keyframe
  • backwards: element will start with the value of first keyframe
  • both: element will retain the value of both keyframe
  • initial: element sets property to default value
  • inherit: element inherits parent property

animation-delay: 2s delays the animation for 2 sec.

Repeating animations

animation-iteration-count: the value can be between 1 to infinite

Direction

animation-direction: decides direction of animation

  • normal: normal animation
  • reverse: reverse animation
  • alternate: normal first, reverse later
  • alternate-reverse: reverse first, normal later
  • initial: initial value
  • inherit: inherits parent animation

Shorthand

animation: (keyframe-name/property) (animation-duration) (animation-timing-function) (animation-iteration-count) (aniamtion-direction)

NOTE: More than one animation can be added by using comma ','
