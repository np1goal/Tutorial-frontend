Javascript BOM (Browser Object Model)

Window

"Browser Object Model" BOM, allows Javascript to talk to browser. The window object is supported by all browsers. All global objects, variables and functions become memebers of the window object. Global variables are properties of the window. Global functions are methods of the window.

window.document.getElementById("demo"); is same as document.getElementById("demo");

Window Size

Two properties can be used to identify the size of browser window; does not include toolbars and scrollbars.

window.innerHeight the inner height of browser window in pixels

window.innerWidth the inner width of browser window in pixels

For IE 8, 7, 6, 5:

document.documentElement.clientHeight OR document.documentElement.clientWidth OR

document.body.clientHeight OR document.bosy.clientWidth

Window Methods

  • window.open() - opens a new window
  • window.close() - close the current window
  • window.moveTo() - move the current window
  • window.resizeTo() - resize the current window

Screen

The window.screen object contains information about the user's screen. window .screen can be written without the window prefix.

Screen Methods

  • screen.width property returns the width of the visitor's screen in pixels.
  • screen.height property returns the height of the visitor's screen in pixels.
  • screen.availWidth property returns the width of the visitor's screen in pixels minus interface features like the Window's Taskbar.
  • screen.availHeight property returns the height of the visitor's screen in pixels minus interface features like the Window's Taskbar.
  • screen.colorDepth property returns number of bits used to display one color.
    • 8 bits: VGA colors
    • 16 bits: High colors
    • 24 bits: True colors --> rgb colors in HTML are True Colors
    • 32 bits: Deep colors
  • screen.pixelDepth property returns pixel depth of screen --> Color Depth and Pixel Depth are equal for modern computers.

Location

The window.location object can be used to get the current page address (URL) and to redirect the browser to a new page. The prefix of window is optional.

  • location.href property returns the URL of the current page.
  • location.hostname property returns name of internet host of the current page like www.w3schools.com.
  • location.pathname property returns path name of the current page like /path/pathname#element.
  • location.protocol property returns web protocol of the current page like https:.
  • location.port property returns number of internet host port of the current page.
  • location.assign() method loads a new document.

History

The window.history object contains the browsers history. The prefix of window is optional.

To protect the privacy of the users, there are limitations to how Javascript can access the this object.

  • history.back() same as clicking back button in the browser
  • history.forward() same as clicking forward button in the browser

Navigator

The window.navigator object contains information about the visitor's browser. The prefix of window is optional.

  • navigator.cookieEnabled returns true if cookies are enabled otherwise false.
  • navigator.appName returns application name of the browser -> "Netscape" is application for IE11, Chrome, Firfox and Safari.
  • navigator.appCodeName returns application code name of the browser -> "Mozilla" is application for IE11, Chrome, Firfox and Safari.
  • navigator.product returns product name of the browser engine -> Do not rely on this!!!
  • navigator.appVersion returns version information of the browser
  • navigator.userAgent returns user-agent header sent by the browser to the server
  • navigator.platform returns for browser platform (Operating System)
  • navigator.language returns browser's language
  • navigator.onLine returns true if browser is online
  • navigator.javaEnabled() returns true if java is enabled

    • Popup Alert

    JavaScript has three kind of popup boxes: Alert box, Confirm box, and Prompt box.

    Alert Box

    An alert box is often used if you want to make sure information comes through to the user. When the alert-box is popped, the user has to click "OK" to proceed.

    alert("Hello") --> window prefix is optional

    Confirm Box

    A confirm box is often used if you want user to verify or accept something. When the confirm-box is popped, the user has to click "OK" or "Cancel" to proceed. If user click "OK", box returns true. If user click "Cancel", box returns false.

    confirm("Hello") --> window prefix is optional

    Prompt Box

    A prompt box is often used if you want user to input a value before entering a page. When the prompt-box is popped, the user has to click "OK" or "Cancel" to proceed. If user click "OK", box returns input value. If user click "Cancel", box returns null.

    prompt("someText", "defaultText") --> window prefix is optional

    Timing

    Javascript can be executed in time-intervals know as timing events.

    Timing Events

    The window object allows execution of a code at specified time intervals called Timing Events. The two key methods are:
    • setTimeout(function, milliseconds) executes a function specified number of elements.
    • setInterval(function, milliseconds) same as setTimeOut() but repeats the execution continously after every specified time.

    How to stop Execution?

    clearTimeout(timeoutVariable) stops the execition of setTimeout()

    clearInterval(intervalVariable) stops the execution of setInterval()

    Cookies

    Cookies store user information in web pages.

    When a browser requests a web page from a server, cookies belonging to the page are added to the request. This was the server gets the necessary data to "remember" information about the users.

    Create a Cookie

    document.cookie = "username: nishthapatel"; This creates a new cookie

    document.cookie = "username: nishthapatel; expires=Thu, 18 Jul 2020 12:00:00 UTC"; This creates a new cookie with an expiry date. By default expiry date is when the browser is closed.

    document.cookie = "username: nishthapatel; expires=Thu, 18 Jul 2020 12:00:00 UTC; path=/"; With path parameter, the browser knows what path the cookie belongs to. By default it belongs to the current page.

    Read a Cookie

    document.cookie; Access the cookie; returns a string of all cookies like: cookie1 = value; cookie2 = value; cookie3 = value

    Update a Cookie

    document.cookie= "username: nish"; Cookies can be overwritten.

    Delete a Cookie

    document.cookie = "username=; expires=Thu, 01 Jan 1970 00:00:00 UTC; path=/;"; To delete a cookie, set the expire date to a past date. To delete the correct cookie, define the correct cookie path.
