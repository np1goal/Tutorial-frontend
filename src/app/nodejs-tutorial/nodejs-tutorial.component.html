Node.js

Introduction

Node.js is an open source server environment which, allows Javascript to run on the server.

Node.js uses asynchronous programming.

Modules

Modules are same as Javascript libraries. A set functions to be included in the application. Node.js has some built-in modules.

There are 3 types of Modules:

  • Core Modules
  • Local Modules
  • Third Party Modules

Include Core Modules

To include a module use require() function

var http = require('http');

Code Module Description
http includes classes, methods and events to create Node.js http server
url includes methods for url resolution and parsing
querystring includes methods to deal with query string
path includes methods to deal with file paths
fs includes classes, methods and events to work with I/O file
util includes utility functions useful for programmers

Create and Include Local Modules

New modules can be created and included in applications. exports keyword is used to make properties and methods available outside the module file.

exports.myDateTime = function() {
  return Date();
};

This can be stored in another file my-module.js

This module can be included in the node file using require()

var myModule = require('./my-module.js');

HTTP Module

Node.js has built-in http module, which allows Node.js to transfer data over Hyper Test Transfer Protocol.

var http = require('http');

Node.js as Web Server

HTTP module can create an HTTP server that listens to server ports and gives a response back to the client.
