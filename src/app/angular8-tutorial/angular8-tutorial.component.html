Angular 8

Introduction

  • Angular app is a modular app. It is made of collection of modules.
  • Every app has a root module named AppModule.
  • Each Module is made of Components + Services

Architecture

Components

  • Root component is AppComponent. AppComponent Module Typescript (app.module.ts) class has NgModule metadata declared in @NgModule tag.
  • A component must belong to NgModule for it to be available to other components.
  • A component is declared in @NgModule.declaration array (aka NgModule metadata).
  • Each Component has HTML (view, template) + Typescript class (logic, data and method) + Metadata (Information, Decorator).
  • @Component is the Component decorator that marks the class to be a Component class. In side decorator:

    • selector: Renders that tag of component to display in template. For example, 'app-root'; It can specified and used in 3 different ways.

      1. Define selector as tag -- selector: 'app-root', use this like <app-root>
      2. Define selector as class -- selector: '.app-root', use this as class = "app-root" in an HTML tag like div
      3. Define selector as attribute -- selector: '[app-root]', use as attribute in an HTML tag like div

    • templateUrl: It is the path to template of component. A small template can be created here also.

      1. Replace templateUrl to template -- template: '<div>This is inline template<div>'
      2. Replace templateUrl to template, use backtics for multiple line template -- template: `<div>This is inline template<div>`

    • styleUrl: It is the path to style sheet of component

      1. Replace styleUrl to styles -- styles: [`div {color: red;}`]

Services

  • Class that contains business logic of the application.
  • It is a class with specific purpose. The purpose could be to share data among multiple compoennts, implement application logic (like calculating age from DOB) or to do external interaction like to connect with Database.
  • Naming convention -- ________.service.ts
  • Services are implemented using dependency injection.

Interpolation

  • Displays the value of a property in TS file. It is written in double curly braces.
  • title displays the value of title in TS file which is {{title}}.
  • Math expressions: result of 2 + 2 is {{2+2}}.
  • String concatenation: result of "This is" + title + "project" is {{"This is" + title + "project"}}.
  • Using JS functions: result of title.length is {{title.length}}.
  • Using JS methods: result of title.toUpperCase() is {{title.toUpperCase()}}.
  • Using JS methods: result of greetUser() method in TS file is {{greetUser()}}.
  • Assignment to a variable is not allowed in interpolation.
  • Property binding with interpolation can only work with string values.

Property Binding

Attributes vs Properties

  • Attributes are defined by HTML vs Properties are defined by DOM (Document Object Model).
  • Attributes initialize DOM properties and cannot be changed vs Properties can be changed.

Class Binding

  • Class Binding

    • In an HTML tag class attribute can be used as propert binding like [class] = "specialText"

    If an HTML tag has both class and [class] then [class] will be executed.

  • Class Binding with conditional expression for one class

    • [class.text-danger] = "hasError" -- if hasError is true than only then textDanger style will be applied.

  • Class Binding with conditional expression for multiple class

    • An object can be written in Ts class, theStyles = { "textSuccess": !this.hasError, "text-danger": this,hasError, "text-special": this.isSpecial }

    [ngClass] = "theStyles" -- ngClass directive binds this to the object theStyles.

Style Binding

[ngStyle] can also be used like [ngClass]

Pipes

Pipes are used in interpolation.
  • String
    • title | lowercase -- makes everything lowercase --> {{title | lowercase}}
    • title | uppercase -- makes everything uppercase --> {{title | uppercase}}
    • title | titlecase -- makes everything titlecase --> {{title | titlecase}}
    • title | slice: 3 -- string from index 3 --> {{title | slice: 3}}
    • title | slice: 3: 8 -- string from index 3 to 8 --> {{title | slice: 3:9}}
    • objectToJson | json -- converts object to json--> {{objectToJson | json}}

  • Numbers
    • 5.23878 | number:'1.2-3' -- 1 integer and 2-3 Rounded decimal places --> {{5.23878 | number:'1.2-3'}}
    • 5.23878 | number:'3.2-3' -- 3 integer and 7-8 Rounded decimal places --> {{5.23878 | number:'3.7-8'}}
    • 0.25 | percent -- gives percentage --> {{0.25 | percent}}

  • Currency
    • 0.25 | currency -- by default dollar --> {{0.25| currency}}
    • 0.25 | currency: 'GBP' -- GBP is for euro --> {{0.25 | currency: 'GBP'}}
    • 0.25 | currency: 'GBP': 'code' -- writes GBP instead of euro sign --> {{0.25 | currency: 'GBP': 'code'}}

  • Date
    • today | date: 'shortDate' -- only date --> {{today | date: 'shortDate'}}
    • today | date: 'shortTime' -- only time --> {{today | date: 'shortTime'}}
    • today | date: 'longDate' -- Full date --> {{today | date: 'longDate'}}
    • today | date: 'longTime' -- Full time with time zone --> {{today | date: 'longTime'}}
