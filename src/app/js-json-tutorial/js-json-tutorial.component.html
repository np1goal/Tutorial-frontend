Javascript JSON

Introduction

JSON (JavaScript Object Notation) is used to store and exchange data.

Exchanging Data

To exchange data between browser and server, the data can only be text. Any JS object can be converted to JSON and sent to server. JSON received from server can also be converted to JS objects.

Sending Data

JSON.stringify() can be used to convert JS object to JSON and then sent.

Receiving Data

JSON.parse() can be used to convert received data into JS object.

Storing Data

Store the received data in any format prefered.

JSON Syntax

  • Data is stored in name/value pairs.
  • Data is seperated by commas.
  • Curly braces hold objects.
  • Square brackets hold arrays.

The name of the value is writtwn with double quotes ("name": "John") unlike JS object.

JSON Values

JSON values must be - string, number, JSON object, array, boolean and null. JS function, date and undefined are not included.

Data Types

{"name": "John"} String are defined in double quotes.

{"age":31} Number must be integer or floating number.

{"employee": {"name": "John", "age": 31}} JSON object in JSON object.

{"names": ["John", "Anna", "James"]} Array in JSON object.

{"values": true} Boolean values can be true or false.

{"name": null} Value can be null.

Parse

Data received from the server is always a string. Parsing data through JSON.parser() becomes Javascript object.

JSON.parser('{"name": "John", "age": 30}');

Parsing Functions

Stringify

Converts JS object into a string.

Stringify Javascript Object

Javascript object can be converted to string.

JSON.stringify({"name": "John", "age": 30})

Stringify Javascript Array

Javascript array can also be converted to string.

JSON.stringify(["John", "Ajay", "Anna"])

Exceptions

In JSON, date objects are not allowed. The JSON.stringify() function will convert any dates into strings.

In JSON, functions are not allowed as object values. The JSON.stringify() function will remove any functions from a JavaScript object, both the key and the value.

To add functions in json convert functions into strings using toString() before running the JSON.stringify() function

NOTE: If functions are sent using JSON, the functions will lose their scope, and the receiver would have to use eval() to convert them back into functions.

Objects

It is a key/value pair. Key specified in double quotes (""), keys and values seperated by colon (:), pairs seperated by comma (,)

Accessing Object

There are two ways to access objects:

myObj = { "name": "John", "age": 30, "car": null };
x = myObj.name; //First way
x = myObj["name"]; //Second way

Looping an Object

for (x in obj) {
  console.log(x + " : " + obj[x]) // x has key, obj[x] has value
}

Nested JSON Objects

Nested object can be access using obj.cars.cars2; OR obj.cars["cars2"];

Modify values

Value of nexted objects can also be modified like obj.cars.cars2 = "BMW"; OR obj.cars["cars2"] = "BMW";

Deleted Object Properties

Use delete keyword like delete obj.cars.cars2;
